Chapter 0

 

In this modern era, everyone favor android app and which has become the part of our daily routine. We are introducing “SASTCollege.apk” the android which holds information about our college and can track the location of college bus. This application can be used through android phone. We describe a network for distributing campus information among teachers, students and employee. The system is used client-server technology to access location and information. For location tracking is used extra one application called bus application. The bus application is installed in driver phone that sends latitude and longitude to the server. It uses the Kalman filter algorithm to provide the correct location of the bus on Google Map.

 

Key words: Android app, College information, Location tracking, Google map, Kalman filter, client-server.
