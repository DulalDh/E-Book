6.2   Limitations

We already mentioned the advantages of college information with bus location tracking in our paper, but apart from the advantages it also has some limitations to use. The main limitations of this system are:

Implementation costs: To implement bus location tracking system is bring a huge financial cost. In this system is used two application such as user application and driver application. So require one android phone for driver application. The android phone is very costly. The cost is  reduce by using tracking device.

System crash occur for interconnection: If there is any damage due to rupturing in connection module then entire system gets crashed. This will be the case of radio signals or the other signals. There may be a problem of signal receiving/sending for gateway, server or application. The server of the system results in crash in most of the systems.

Network problem: Scenario of our country network system is very bad. It is very common to fall of network in various area of our country. As our entire system fully dependent on internet network, so that if there is no internet the system will fail.

Human errors: human errors may occur, if the human does not handle the application safely or does not provide the correct input to perform the operations. Human errors also lead to destructions of the system. Then there will be a huge loss of  time.

Reliability: It seems that, the reliability of the college information with bus location tracking system may vary in different situation. It depends mostly on the technology used and the advancements being done or methodology being used.
